ISREAL Authentication

A comprehensive authentication system.

Introduction

This project utilizes a robust database system to store user credentials and other relevant information. For development purposes, we use a local database stored on the device.

We leverage the power of Alchemy to streamline our database operations.

On the backend, we employ Python and Flask to create a scalable and efficient server-side application.

For the client-side, we utilize HTML, CSS, and JavaScript to create a seamless and interactive user experience.
